# Environment Details
The core dependencies for the ConvAHKG model are:

*   Deep Learning Framework: tensorflow==2.17.0 and keras==3.4.1 (using the Keras 3 multi-backend).

*   Hyperparameter Optimization: optuna==3.6.1 and keras-tuner==1.4.7.

*   Scientific Computing: numpy==1.26.4, scipy==1.13.1, pandas==2.2.2, scikit-learn==1.5.1.

*   Natural Language Processing: nltk==3.8.1, gensim==4.3.3.

*   GPU Acceleration: Required NVIDIA CUDA Toolkit 12.3 and associated libraries (as listed in the environment).
